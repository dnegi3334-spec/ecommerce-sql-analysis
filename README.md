# 🛒 E-Commerce Sales Analysis — SQL Project

A complete end-to-end SQL project analyzing e-commerce sales data 
to uncover revenue trends, customer behaviour, and product performance.

## 📌 Tools Used
- PostgreSQL
- pgAdmin 4

## 🗂️ Dataset
- 5 Tables: customers, products, orders, order_items, categories
- 30 Customers | 20 Products | 50 Orders
- Time Period: 2022–2024

## 📊 Analysis Performed
- Basic: COUNT, GROUP BY, WHERE, EXTRACT
- Medium: JOINs, Subqueries, HAVING, CASE WHEN
- Advanced: CTEs, Window Functions (RANK, LAG), Running Totals

## 💡 Key Insights
- Electronics drives highest revenue
- 40% customers are repeat buyers
- Maharashtra, Delhi & Karnataka are top revenue states

## ▶️ How to Run
1. Run `setup.sql` to create tables and insert data
2. Run `analysis_queries.sql` for all 25 queries
